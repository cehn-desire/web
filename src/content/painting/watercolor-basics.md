# 水彩入门：从第一笔开始

> 水彩的魅力在于它的不可控——水与颜料在纸上的交融，总会带来意想不到的惊喜。

## 为什么选择水彩

水彩画有着其他媒介无法替代的特点：

- **透明感**：颜料层层叠加，底层色彩透出，形成丰富的层次
- **流动性**：水的参与让画面充满偶然性和生命力
- **便携性**：一套便携水彩 + 一本水彩本，走到哪画到哪
- **快速干燥**：相比油画，水彩干得很快，适合快速写生

## 入门工具

### 基础配置

```
必备清单：
├── 水彩颜料（管状或固体）
│   ├── 推荐：温莎牛顿、史明克学生级
│   └── 核心颜色：8-12 色足够
├── 水彩笔
│   ├── 圆头笔 #6, #10
│   └── 平头笔 1/2 inch
├── 水彩纸
│   ├── 棉浆纸（推荐）
│   ├── 克重：300g/m²
│   └── 纹理：中粗纹 (Cold Press)
└── 辅助工具
    ├── 调色盘
    ├── 洗笔筒
    └── 吸水海绵/纸巾
```

## 基本技法

### 1. 平涂 (Flat Wash)

将画板倾斜约 15°，用大笔均匀涂刷。

### 2. 渐变 (Graded Wash)

从浓到淡的自然过渡，关键是控制水量。

### 3. 湿画法 (Wet-on-Wet)

```text
步骤：
1. 用清水打湿纸面
2. 等待纸面水分被吸收（不再反光）
3. 点上颜料，观察色彩扩散
```

湿画法适合画天空、远山、水面等需要柔和过渡的部分。

### 4. 干画法 (Wet-on-Dry)

在干燥的纸面上作画，笔触清晰锐利，适合细节刻画。

## 练习建议

1. **色卡练习**：熟悉每个颜色的特性
2. **渐变练习**：掌握水量控制
3. **单色画**：先用一个颜色画明暗关系
4. **每日一张**：坚持比天赋更重要

水彩是一门需要耐心的艺术。别怕画坏，每一张"失败"的作品，都是通往进步的阶梯。
